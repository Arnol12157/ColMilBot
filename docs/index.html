Get started

First, you need to install node.js. Verify that you are running at least node 6.9.x and npm 3.x.x by running in a terminal/console window: 

node -v
npm -v

Older versions could produce errors, but newer versions are fine.

Install project dependencies:

npm install

After this project is ready for development.

Build project:

npm run build

Serve project:

npm run serve

Project is using Angular CLI. You can install it for additional features.

Deployment

To deploy project to server, you need to build it:

npm run build

After that copy content of dist folder to your server.

Your server should be configured properly to handle Angular routing functionality.

For example, Apache server: add a rewrite rule to the .htaccess file:

RewriteEngine On
# If an existing asset or directory is requested go to it as it is
RewriteCond %{DOCUMENT_ROOT}%{REQUEST_URI} -f [OR]
RewriteCond %{DOCUMENT_ROOT}%{REQUEST_URI} -d
RewriteRule ^ - [L]

# If the requested resource doesn't exist, use index.html
RewriteRule ^ /index.html

You can find more at Angular documentation

Material Design

Material Design is a unified system that combines theory, resources, and tools for crafting digital experiences.

We created MOMO admin template guided by Material Design principles. Material Design has awesome specification.

Community proposes many implementations of Material Design.

We are using in our project Material Design for Angular. This framework is built by Angular team. Show case with examples.

Also we are using Covalent. Covalent is UI platform based on Angular Material. Show case with examples.

Angular Material and Covalent are in active development and have many stars at GitHub.

Styles and themes

We are using SASS for generating styles for project ang Angular emulated view encapsulation. This mode emulates the behavior of shadow DOM by preprocessing (and renaming) the CSS code to effectively scope the CSS to the component's view. You can find more here.

Angular Material and Covalent support themes by default. Angular Material guide and Covalent guide have good description how create your own theme.

Themes initialization code is located at src/styles/theme/theme.scss. If you need to set custom colors, look at src/styles/_variables.scss

Covalent has many helpful styles, find description here
